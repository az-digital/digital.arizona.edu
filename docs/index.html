Latest Versions

Arizona Bootstrap

Bootstrap 4 New

Arizona Bootstrap is the University of Arizona branded version of Bootstrap 4. This version of Bootstrap will be integrated into the Quickstart 2.0 theme.

Developers and site owners utilizing the previous version of Bootstrap can expect an upgrade path into Arizona Bootstrap, and backwards compatibility with select components and utilities.

Explore Arizona Bootstrap

Quickstart 2.0

Drupal 9 New

A new version of the University's preconfigured version of Drupal is here! Quickstart 2.0 (AZ Quickstart) leverages the power of Drupal 9 combined with Arizona Bootstrap to provide a foundation for developers building Drupal-based sites.

Quickstart 1.0 developers and site owners can expect an upgrade path from Quickstart 1.0 to Quickstart 2.0.

Get Started with Quickstart 2.0

Bootstrap 3 Legacy

UA Bootstrap is the University of Arizona branded version of Bootstrap 3. This version of Bootstrap is integrated into the Quickstart 1.0 theme. Extending UA Bootstrap will allow you to quickly build responsive, mobile-first projects on the web.
