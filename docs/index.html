Latest Versions

Arizona Bootstrap 5

Alpha

Arizona Bootstrap 5 is a University of Arizona branded version of Bootstrap 5. This alpha release offers new features, improved grid and utility systems, and better accessibility support.

This version is in active development and is intended for testing and feedback before it is integrated into Quickstart 2.

Try Arizona Bootstrap 5

Arizona Bootstrap 2

Stable

Arizona Bootstrap 2 is the University of Arizona branded version of Bootstrap 4. It is stable and production-ready, and is integrated into the Quickstart 2 theme.

Developers can extend this version or prepare for migration to Bootstrap 5 using the same design language and components.

Explore Arizona Bootstrap 2

Quickstart 2

Drupal 10

Quickstart 2 (AZ Quickstart) is the University’s current Drupal 10 distribution. It combines Arizona Bootstrap 2 with best practices for building accessible, scalable sites.

Includes features like Paragraphs, Layout Builder, and configuration defaults for UA-branded content.

The az_migration module provides an upgrade path from Quickstart 1.0.

Legacy Versions

UA Bootstrap

Bootstrap 3 Legacy

End of Life Warning

UA Bootstrap will no longer be supported by Arizona Digital after November 15th, 2023. It is critical to get all sites and applications updated to Arizona Bootstrap before this date. For assistance, reach out on the #azdigital-support channel on Slack.

UA Bootstrap was the University of Arizona’s version of Bootstrap 3, integrated into the Quickstart 1.0 theme.
