Arizona Bootstrap

Build responsive, mobile-first projects on the web with the University of Arizona's theme for Bootstrap.

Bootstrap is an open source toolkit for developing with HTML, CSS, and JS. Quickly prototype your ideas or build your entire app with our Sass variables and mixins, responsive grid system, extensive prebuilt components, and powerful plugins built on jQuery.

A team of web-focused volunteers known as Arizona Digital meets weekly to build and test products like Arizona Bootstrap and Arizona Quickstart. Contributions are what keep Arizona Bootstrap moving and improving, and are in everyone's best interest.

Installation

Include Arizona Bootstrap’s source Sass and JavaScript files via npm, Composer or Meteor. Package managed installs don’t include documentation, but do include our build system and readme.

npm install https://github.com/az-digital/arizona-bootstrap

Arizona Digital CDN

When you only need to include Arizona Bootstrap’s compiled CSS or JS, you can use the Arizona Digital CDN.

CSS only
<link rel="stylesheet" href="https://cdn.digital.arizona.edu/lib/arizona-bootstrap/2.0.20/css/arizona-bootstrap.min.css" crossorigin="anonymous">
JS, Popper, and jQuery
<script src="https://cdn.jsdelivr.net/npm/jquery@3.6.1/dist/jquery.slim.min.js" integrity="sha384-MYL22lstpGhSa4+udJSGro5I+VfM13fdJfCbAzP9krCEoK5r2EDFdgTg2+DGXdj+" crossorigin="anonymous"></script>
